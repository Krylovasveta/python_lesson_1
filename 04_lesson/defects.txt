1. Заголовок:
Если строка начинается с пробела, функция capitalize не преобразовывает первую букву строки заглавной
Шаги:
1. Импортировать класс StringUtils
2. Вызвать функцию capitalize с параметром, начинающаяся с символа пробел, например, '   hello'.

Ожидаемый результат:
Первая буква станет заглавной

Фактический результат:
Первая буква осталась строчной (пр. '   hello')

Окружение: Windows 7, VS Code 1.114.0, Python 3.14.2,  Pytest 9.1.1

2.Заголовок:
Функция capitalize делает все буквы, кроме первой, строчными
Шаги:
1. Импортировать класс StringUtils
2. Вызвать функцию capitalize с параметром, содержащим заглавную букву, например
 hi, i am Sveta

Ожидаемый результат:
Первая буква станет заглавной, остальные буквы останутся в том же регистре

Фактический результат:
Заглавная буква станет строчной (пр. 'hi, i am sveta')

Окружение: Windows 7, VS Code 1.114.0, Python 3.14.2,  Pytest 9.1.1
